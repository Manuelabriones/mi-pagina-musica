#  Mi Página de Música

Este proyecto es una página web básica creada con HTML. El objetivo es mostrar mis gustos musicales, incluyendo géneros y artistas favoritos.

## Características

- Uso de etiquetas semánticas HTML
- Imágenes de artistas
- Enlaces de navegación
- Formulario de contacto
- Página publicada con GitHub Pages
