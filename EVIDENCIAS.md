## Aprendizajes

### 1. ¿Qué fue lo más fácil y lo más retador?
Lo más fácil fue crear la estructura HTML. Lo más retador fue conectar el repositorio con GitHub y publicar el sitio.

### 2. ¿Qué etiquetas semánticas usaste y por qué?
Usé header, section, article y footer para organizar mejor el contenido.

### 3. ¿Cómo organizaste tus commits?
Realicé commits por cada cambio importante, como crear el index.html, agregar imágenes y crear el README.

### 4. ¿Qué mejorarías?
Agregaría mas estilos CSS para mejorar el diseño y hacerlo más atractivo.
